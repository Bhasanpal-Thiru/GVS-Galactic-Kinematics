# Holographic Geometric Vacuum Scaling (GVS)
Research repository for the paper: "Holographic Geometric Vacuum Scaling (GVS): An Empirical and Kinematic Model for Galactic Rotation Curves."
[![DOI](https://zenodo.org/badge/1177144457.svg)](https://doi.org/10.5281/zenodo.18928146)

## Contents
- `GVS_Analysis_Main.ipynb`: Full MCMC fitting pipeline.
- `GVS_Results_Master.csv`: Derived alpha-couplings and kinematic triggers for 175 SPARC galaxies.
- `Figures/`: High-resolution PDFs of the holographic correlation, density profiles, and sample fits.

## Data Source
This project utilizes the SPARC (Spitzer Photometry and Accurate Rotation Curves) database (Lelli et al. 2016).
